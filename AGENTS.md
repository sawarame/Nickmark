# Project Gemini Instructions (Nickmark)

このファイルは Nickmark プロジェクトの基本的なガイドラインと命令を定めたものです。
開発にあたっては、以下の `.context` フォルダ内にある詳細なドキュメントを必ず参照し、その指示に従ってください。

## 基礎ドキュメント

- **[仕様書 (.context/spec.md)](.context/spec.md)**
  - プロジェクトの基本コンセプト、機能要件、データ構造について記述されています。
- **[技術スタック (.context/tech_stack.md)](.context/tech_stack.md)**
  - 使用されているライブラリ、ツール、フレームワークの選定理由と構成が記述されています。
- **[開発ワークフロー (.context/development.md)](.context/development.md)**
  - 環境構築、ビルド、テスト、リリースの手順について記述されています。

## コーディング・デザイン規約

- **[TypeScript コーディングルール (.context/typescript.md)](.context/typescript.md)**
  - 型定義、命名規則、コンパイラ設定などの実装ルールが記述されています。
- **[デザインルール (.context/design.md)](.context/design.md)**
  - UI/UX の原則、コンポーネント設計、スタイルガイドが記述されています。

## 優先順位

1. ユーザーからの直接の指示
2. 本ファイル (`GEMINI.md`) の指示
3. `.context/` フォルダ内の各ドキュメントの指示
